# The Scene Co. Live — Premium Landing Page

This repository contains the premium, launch-ready website for **The Scene Co. Live**, an event management and production company that creates premium, immersive live experiences.

The design utilizes a **dark luxury aesthetic**, incorporating dynamic ambient stage-lighting glows in blush pink, violet, and electric blue, editorial-magazine grids, a digital film grain overlay, and typography pairing `Bodoni Moda` and `Inter` (Metropolis style).

## Features & Sitemap

The website contains all 11 premium sections structured in a high-fidelity narrative:
1.  **Navigation Header**: Floating, glassmorphic design with backdrop blurs and an action CTA.
2.  **Hero Section**: Immersive atmosphere with glowing light leaks, strong serif headers, and action triggers.
3.  **About Section**: Sophisticated brand statement detailing spatial engineering, paired with interactive stats counters.
4.  **Services Section**: Grid of 9 premium event capabilities (Corporate, Wedding, Activations, Concerts, etc.) with custom hover states.
5.  **Featured Projects (Portfolio)**: A letterboxed 21:9 grid featuring 6 case studies with smooth hover animations.
6.  **Why Choose Us**: 6 pillars of execution standards highlighted by blush pink numbers.
7.  **Process timeline**: A linear vertical framework mapping the project lifecycle from Discovery to Wrap-up.
8.  **Testimonials**: 4 luxury quotes from verified collaborators.
9.  **FAQ**: Accordion-based answers addressing common client concerns (pricing, scope, permits, etc.).
10. **Inquiry Form**: An elegant, qualification-focused contact form for lead generation.
11. **Footer**: Directory columns, socials, and legal credits.

## Local Development

This project uses **Vite** for rapid local serving and production bundling.

### 1. Installation
Install the development dependencies:
```bash
npm install
```

### 2. Run Local Server
Start the hot-reloading development server:
```bash
npm run dev
```

### 3. Build Production Bundle
Compile the optimized static site:
```bash
npm run build
```
